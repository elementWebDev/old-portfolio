Centering Content

There's 2 main ways to center things, but the CSS you write depends on what you're centering.

If: you want to center an entire block-level tag

And: it is fixed-width

Then: set the left and right margins to auto

If: you want to center children inside a block-level tag

Then: set the text-align of the children to center
