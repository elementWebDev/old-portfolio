Grooming Your Code

Level 4

  • Staying DRY

In this section, we're going to talk about different ways that you can clean up your CSS, including Staying DRY, knowing and understanding display types, and a little bit about horizontal centering.

One of the fundamental ways that you can clean up your CSS is to be DRY.

  • D - Don't
  • R - Repeat
  • Y - Yourself

In programming, the DRY acronym stands for don't repeat yourself, as it's a practice that will save you tons of time in declaring and maintaining your styles.
